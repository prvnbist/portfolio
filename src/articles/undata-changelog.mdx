## Undata Changelog

undata, an application that lets you query your database and populates the result in a table and more to come. Built with NextJs 14, MantineUI, TRPC w/ React Query, Monaco Editor and React Table.

### v0.0.5 (2024-02-08)

Added custom cell renderers to the table component

-  Link Cell
-  Single Select Cell
-  Image Cell
-  JSON Cell
